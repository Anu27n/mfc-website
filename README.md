## Mozilla Firefox Club Starter Template

## How to get started
1. Download/clone the starter template
2. Open in VS Code -> Open Terminal
3. Run the following commands in the terminal<br>
   ```yarn install```<br>
   ```yarn run dev```

## Features
Tailwind CSS<br>
Material-UI<br>
Roboto font<br>
<br><br>
Frontend: Vite 4.2.0<br>
Package manager: Yarn
